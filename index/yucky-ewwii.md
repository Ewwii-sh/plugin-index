Yucky Ewwii is an **official** ewwii plugin that provides yuck (eww) compatibility. It is useful for users migrating from eww 
or for users who prefer lisp-like languages in general.

- GitHub: https://github.com/Ewwii-sh/yucky-ewwii
- Website: https://ewwii-sh.github.io/yucky-ewwii
